Download instructions:

Eclipse

First of all, in order to install Acceleo, you need Eclipse. We are recommending the installation of the latest version of the modeling bundle available in the download page.

After having unzipped it, you can tweak the settings of Eclipse to be more confortable. In order to that, you have to edit the "eclipse.ini" file and add or change the following settings:

  • -Xmx1024m
  • -Xms256m
  • --launcher.XXMaxPermSize 256m
With those settings, Eclipse will be significantly faster. You can install another version of Eclipse if you want but since the Eclipse modeling package provides a basic environment for model driven development, it is the most suited for a code generation project.

Update sites

The easiest (and quickest) way of installing Acceleo is to use the dedicated update site. Start by downloading and installing your preferred Eclipse distribution from the Eclipse download page. Then go to the help menu to install a new software.

Then in the upper right part, click on "add" and type the location of the update site desired. You can find the list of the available update site in the download page.

You see here the result with the update site of the 3.1 branch of Acceleo.

You can then select all the Acceleo components for a full install. You can choose to install only the Acceleo runtime if you only want the generation engine to launch Acceleo generator and not the Acceleo tooling to create Acceleo generators. Click Finish, read and accept the Eclipse Public License, and watch Acceleo install itself along with its dependencies.

Marketplace

In order to install Acceleo thanks to the Eclipse Marketplace, first download an Eclipse bundle with the Eclipse Marketplace like the Eclipse Modeling bundle. Then open the Eclipse Marketplace.

Find Acceleo, select it and click finish.

Discovery UI

In order to use the discovery UI, you have to download first the Eclipse Modeling bundle (see install Eclipse). Then open the discovery UI.

Then select Acceleo and click finish.

Direct Download

You can download the Acceleo zipped update site from the Eclipse servers here. Then, open the help menu and select add a new software.

Click the "add..." button and select "archive" then select the zipped Acceleo update site downloaded previously. Select the Acceleo SDK, click finish and you're done!
