Source Code

Download from the Eclipse repository

Following the release of Acceleo 3.1, all the Acceleo source code has been transferred from CVS to Git. The Acceleo source code is now available on the Git repository of the Eclipse Foundation.

  • git://git.eclipse.org/gitroot/m2t/org.eclipse.acceleo.git
  • ssh://git.eclipse.org/gitroot/m2t/org.eclipse.acceleo.git
  • http://git.eclipse.org/gitroot/m2t/org.eclipse.acceleo.git
You can track the work on Acceleo on this page.

Fork on Github

Now that the Acceleo source code is available on Git, a mirror has been created on the official account of the Eclipse Foundation on Github. You can fork the Acceleo source code on github by using this repository.

Generators

Publish your generators on the Obeo Network

The Acceleo community offers a dedicated marketplace for model driven technologies including Acceleo. If you wish to distribute one of your Acceleo generator, you can take advantage of the OBeo Network to distribute your generators to the Eclipse community through the Obeo Marketplace. For more information, have a look at the Obeo Marketplace.

Bugs

Eclipse bugzilla

Acceleo uses the bugzilla of the Eclipse Foundation thus you can find the opened bug entries for Acceleo here. You can open a new bug for Acceleo using the form available here (Acceleo is available in the Modeling category and the M2T component).

Mailing list

Model to Text Mailing List

The Acceleo developper mailing list is available here. You can find there the archive of all the message sent on this mailing list. This mailing list should only be used to contact Acceleo developpers with a question or an issue regarding the source code of Acceleo (if you want to contribute to Acceleo for example). Most of the issues should be raised on the Model to Text (M2T) forum of the Eclipse Foundation monitored by the Acceleo dev' team.
