Presentations and videos: showing Acceleo to the world!

Presentations

Code generation

The following presentation of Acceleo 3.1 and Obeo Traceability 2.0 has been given during Code Generation 2011. This presentation requires the browser Chrome in order to be viewed because it has been created using some of the latest features available in HTML5 that are only supported by Chrome and Firefox but because the videos in this presentation are encoded in mp4 the videos cannot be watched in Firefox.

Acceleo at Code Generation 2011

Topcased days 2011

In this video, you will be able to discover all the features available in Acceleo 3. We will see how to manage a code generation project with Acceleo. From the initialization of your generators to their deployment and their debugging, you will see how to Acceleo can help you to handle the problem that you can encounter. This presentation also features the first public presentation of the new version of Obeo Traceability, an Eclipse based tool used to improve the management of Acceleo generation.

Acceleo at the Topcased Days 2011

Eclipsecon 2010

The Android tutorial will use the prototype based approach in order to realize an Android generator. We will start with a small Android application that will be used to parameterized an Acceleo generator that we will improve so as to transform it into a generic Android application. Then we will create different model of Android applications that will be generated without any changes to our generator. Finally, we will see how to seamlessly integrate this generator within Eclipse.
