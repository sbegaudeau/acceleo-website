Tutorials: step by step tutorials for everyone.

UML to Java

General purpose code generation

The UML to Java tutorial is the most complete tutorial available for Acceleo 3.1.1. You will start with the installation of the development environment and after a review of the concepts used in an Acceleo generator, you will quickly move to the creation of your first Acceleo generator. You will see how to create a good generator following the best practices recommended by the Acceleo dev' team and you will see how to prepare an UI for your generator and how to prepare the generator to use it out of Eclipse. Finally you will see how easy it is to deploy your generator in another Eclipse to integrate code generation to your own tools seemlessly. This tutorial is a step by steps guide accross all the stages of the lifecycle of a code generation project with screenshots and the source code of the result.

Start the tutorial now!

Domain specific model to HTML

Generating from domain specific models

You want to go a bit further in model driven development by creating your own domain specific models and you want to generate code from them? Acceleo fully support the domain specific model approach and in this tutorial, you will create models to represents your need and then generate production ready source code.
